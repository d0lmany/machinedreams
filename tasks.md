## 1. Задача: «Погода и одежда»
**Описание:** Создай программу, которая подскажет, во что одеться, в зависимости от температуры на улице.  
**Требования:**
- Пользователь вводит температуру с клавиатуры. 
- Если температура выше 20°C — вывести: "На улице тепло, можно надеть футболку." 
- Если температура от 0 до 20°C — вывести: "Прохладно, лучше взять куртку." 
- Если температура ниже 0°C — вывести: "Холодно! Одевайся потеплее!" 

Пример работы программы:
```text
Введите температуру на улице: -5
Холодно! Одевайся потеплее!
```

## 2. Задача: «Меню ресторана»
**Описание:** Создай программу, которая управляет меню ресторана. Она должна:
- хранить список блюд, 
- давать возможность добавлять и удалять блюда, 
- показывать текущее меню.

Пример работы программы:
```text
Добро пожаловать в ресторан!
Что хотите сделать?
1 - Посмотреть меню
2 - Добавить блюдо
3 - Убрать последнее блюдо
4 - Выход
Выберите действие: 2
Введите название блюда: Суп
Блюдо добавлено!

Выберите действие: 1
Текущее меню:
1. Суп
```
**Требования:**
- Используй список как структуру данных. 
- Реализуй меню действий с помощью цикла while. 
- При добавлении используй append(), при удалении — pop().

## 3. Расчёт стоимости основных производственных фондов
**Описание:** Дано ОПФ: ввод активов, вывод, дата ввода/вывода, а также ОПФ в начале года. Программа должна:
- рассчитывать количество месяцев от заданной даты ввода/вывода
- рассчитать среднегодовую СОПФ
- вывести итоги

**Требования:**
- ОПФ - объект, для реализации использовать классы.
- количество вводимых ОПФ должно быть не ограничено.

## 4. Калькулятор систем счисления
**Описание:** Создай программу, которая позволяет пользователю конвертировать числа между двумя системами счисления (например, десятичную и двоичную).

**Требования:**
- Реализуйте функции для конвертации числа из одной системы счисления в другую.
- Пользователь должен иметь возможность выбрать систему счисления для ввода и вывода.
- Основание системы счисления должно быть от 2 до 60.

## 6. Поиск второго по величине
**Описание:** Дан массив чисел. Найди второе по величине число (без сортировки и без встроенных функций).

Пример работы программы:
```text
[3, 1, 4, 1, 5] → ответ: 4
[7] → ответ: null или "нет второго"
[2, 2, 2] → ответ: null (нет различного второго)
```
